# SpeedDating
스피드데이팅 애플리케이션

### 구현한 기능
- [x] 회원가입
- [x] 로그인
- [x] 기본 정보 수정
- [x] 디테일 정보 수정
- [x] 이상형 설정
- [x] 외부 링크 연결
- [x] 채팅방 각각 생성
- [x] 일대일 채팅
- [x] 익명 채팅
- [x] 시그널 및 답장

### 실행 방법
1. constants/firebaseConfig.js를 작성한다.
2. `expo start`
